> **「旅人よ、何があった……？」**

## 🛡️ Introduction: 鉄の斧と、電子の魔導書

シュタークは、途方に暮れていた。

村の重要な水源を汚す「魔族（課題）」が現れた。物理的な斧では太刀打ちできない。
「AIは魔法のランプだ」と教わったが、画面に「助けてくれ」と打ち込んでも、返ってくるのは冷たい環境規制のガイドラインだけ。

「伝え方（プロンプト）」がわからない。

力が足りないんじゃない。自分の想いを、AIという巨大な機関が理解できる「呪文」に変換する方法を知らないだけなのだ。

昔は、魔法の書を覚えればなんとかなると思っていた。
今は、パソコンを眺めれば何かできると思っていた。
……どちらも、動かし方がわからなければ、ただのガラクタだ。

そんな彼の前に、一冊の古びた、しかし新しい輝きを放つ魔導書が差し出された。

**『Neu_Zoltraak_Traktat（ノイ・ゾルトラーク・トラクタート）』**

これは、あなたの「混沌（モヤモヤ）」を、世界を変える「一般攻撃魔法（MVPプロンプト）」へと精錬する物語である。

---

## 🔮 この魔導書の「術式」（機能）

1. **認知の初期化（ショーモン）：** 最初の一言で、あなたの課題を「知・情・意」の最適ルートへ導く。
2. **三章の巡礼：**
   - **📘 青の章：魔法使い（知）** — 現状を整理し、解決すべき「問い（What）」を見出す。
   - **📕 赤の章：戦士（意）** — ターゲット（Who）を射抜き、実行の武器を研ぐ。
   - **📗 緑の章：僧侶（情）** — 継続の根（Core Value）を植え、安心を届ける。
3. **真実の試練：** 抽象的な「綺麗事」を検知し、具体的・肉声的な言葉へ引きずり戻す（※将来の拡張予定）。
4. **叡智の結晶化：** 最後に、汎用AIで即座に使える **MVPプロンプト** を授ける。

## 🚀 使い方

1. `index.html` をブラウザで開く。
2. 魔導書の問いに、素直な今の気持ちをぶつける。
3. 三人の賢者との対話を通じて、自分の「相棒」を形にする。
4. 完成した「最初の魔法（MVPプロンプト）」をコピーし、ChatGPTやClaudeに貼り付ける。

## 📝 Architecture

このアプリは、Y Combinator式のピッチ分析ワーク（Who / What / Core Value）を、物語体験（ヒーローズ・ジャーニー）として再構築したものです。

「説明される」のではなく「体験する」ことで、AI駆動の開発に必要なマインドセットを自然に獲得することを目指しています。

### 章の順番が変わる仕組み

オープニングの一問（「自分が困っている／誰かが困っている／世界をよくしたい」）によって、3章の順番が動的に変わります。ユーザーは気づかないまま、最も答えやすい順番で進んでいく設計です。

| オープニング選択 | 章の順番 |
|---|---|
| 自分が困っている | 緑 → 青 → 赤 |
| 誰かが困っている | 赤 → 青 → 緑 |
| 世界をよくしたい | 青 → 赤 → 緑 |

### 各章の選択肢

| 章 | テーマ | 選択肢 |
|---|---|---|
| 📘 青の章 | AIの性格・機能 | 魔法使い型 / 戦士型 / 僧侶型 |
| 📕 赤の章 | 誰のため | 自分自身 / 特定の誰か / 不特定多数 |
| 📗 緑の章 | 届ける価値 | 気づき / 行動 / 継続 |
